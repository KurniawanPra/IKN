# Nusantara Rubber Industry - Company Profile & E-commerce Website

A professional Next.js 14 website for Nusantara Rubber Industry featuring company profile and e-commerce functionality with modern UI/UX design.

## Features

- **Modern Design**: Glass morphism effects, smooth animations, and professional layout
- **Responsive**: Fully responsive design that works on all devices
- **Company Profile**: Comprehensive information about the company, history, and values
- **E-commerce Functionality**: Product catalog, detailed product pages, and shopping cart
- **Performance Optimized**: Built with Next.js 14 for optimal performance and SEO
- **Multi-language Support**: Ready for Indonesian and English languages
- **Accessibility**: Designed with accessibility best practices

## Technology Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with custom glass morphism and animation utilities
- **Animations**: Framer Motion for smooth transitions and animations
- **TypeScript**: For type safety and better developer experience
- **Icons**: Custom SVG icons for lightweight performance

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

```
src/
├── app/                 # Next.js 13/14 app router pages
├── components/          # Reusable UI components
├── public/              # Static assets (images, icons, etc.)
└── styles/              # Global styles and CSS
```

## Key Components

- **Navbar**: Responsive navigation with language toggle
- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Company history, mission, and values
- **Products Section**: Product catalog with e-commerce functionality
- **Features Section**: Company advantages and differentiators
- **Testimonials Section**: Customer feedback and partnerships
- **Newsletter Section**: Email subscription form
- **Contact Section**: Contact information and inquiry form
- **Footer**: Comprehensive footer with links and social media

## Customization

### Colors
The website uses a professional color scheme based on indigo and violet gradients, representing trust, quality, and innovation.

### Typography
Clean, readable typography optimized for both Indonesian and English content.

### Images
Replace placeholder images in `/public/images/` with actual product and company photos.

## Deployment

The application can be deployed to any platform that supports Next.js applications:
- Vercel (recommended)
- Netlify
- AWS Amplify
- Docker containers
